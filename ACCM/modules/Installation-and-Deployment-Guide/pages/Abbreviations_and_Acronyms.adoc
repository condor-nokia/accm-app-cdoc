= Abbreviations and Acronyms

The following table lists the acronyms used in this guide.
Note that some of these acronyms might have alternate definitions.

[width="100%",cols="27%,73%",options="header"]
|===
|*Acronym* |*Definition*
|AAC |Altiplano Access Controller
|ACCM |Access Cluster Campaign Manager
|API |Application Programming Interface
|AV |Altiplano Virtualizer
|CIR |Committed Information Rate
|CLA |Closed Loop Automation
|CPE |Customer Premises Equipment
|CPU |Central Processing Unit
|IBN |Intent-Based Networking
|IPFIX |Internet Protocol Flow Information Export
|LS |Lightspan
|LT |Line Termination
|NBI |Northbound Interface
|OLT |Optical Line Termination
|ONT |Optical Network Termination
|ONU |Optical Network Unit
|OpenTSDB |It is a scalable time series database that simplifies the process of storing and analyzing large amount of data.
|OSS |Operations Support System
|PM |Performance Monitoring
|UI |User Interface
|URL |Uniform Resource Locator
|VNO |Virtual Network Provider: used with the same meaning as ISP and CSP in this context
|WebUI |Web User Interface
|===
