= How to use {app-short-name} as {vno-initials}

When the {app-short-name} is installed on top of an Altiplano solution, the {vno-initials}s must act through the proxies
and the WebUI which the application offers.
These proxies and WebUI applies all the rules configured in {app-short-name}.
Refer to xref:How_to_use_WNP_as_INP.adoc[How to Use {app-short-name} as {inp-initials}] section to know the
available configurations.
